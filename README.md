# Internal Project Management SPA

## Description
A lightweight, secure, and fully responsive Single Page Application (SPA) built using modern Vanilla JavaScript module structure. It manages corporate assignments across enterprise levels, ensuring strict view isolation and field level edits using a clear role paradigm matrix (Managers and Collaborators).

## Technologies
* HTML5 Structural Semantics
* JavaScript (ES6 Modules & Fetch Async API Wrapper)
* Tailwind CSS Integration CDN Engine
* JSON-Server API Framework Mock Architecture

## Installation
1. Clone or copy your project files locally into a workspace directory.
2. Ensure you have Node.js installed on your machine.
3. Install the mock data provider globally or within your local node dependency tree:
```bash
   npm install -g json-server
